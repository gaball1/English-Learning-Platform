# Educational English Learning Game for Kids  

An exciting Unity-based game designed to make learning English interactive, fun, and rewarding for children. This project was developed as part of the **Computer Animation** course by a dedicated team of passionate developers.  

## ðŸŒŸ Key Features  

### Main Menu  
- Simple and user-friendly navigation for kids.  

### Character Selection  
- Children can choose their favorite character to accompany them throughout their learning journey.  

### Alphabet Learning Level  
- Each letter is paired with a video showing words starting with that letter.  
- Fun and educational songs enhance the learning experience.  

### Numbers Learning Level  
- Numbers are introduced with real-world examples.  
- Catchy tunes make number learning enjoyable and memorable.  

### Quizzes  
- Scoring system to track performance.  
- Retry options for improving scores.  
- Time tracking adds an exciting challenge (time left or time taken).  

### Two Mini-Games  
1. **Sword Game**:  
   - Letters appear with matching images.  
   - Kids hit the correct letter with a sword to earn points.  

2. **Matching Game**:  
   - Connect shapes and letters to boost cognitive skills and letter recognition.  

### Sound Effects and Background Music  
- Engaging audio keeps kids motivated and immersed in the game.  

### Certificate of Achievement  
- At the end of the game, children receive a personalized certificate featuring their chosen character, celebrating their success!  


## ðŸŽ® How to Play  

1. Launch the game and navigate through the main menu.  
2. Select a character and start exploring the different levels:  
   - Learn the alphabet with videos and songs.  
   - Understand numbers with examples and catchy tunes.  
3. Test your knowledge with exciting quizzes.  
4. Enjoy the mini-games to reinforce learning and have fun!  
5. Earn a certificate at the end to celebrate your achievement.  

## Technologies Used  

- **Unity**: Game development and design.  
- **C#**: Game logic and interactivity.  

## Installation  

1. Clone the repository:  
    ```bash
    git clone https://github.com/gaball1/English-Learning-Platform.git
    ```  
2. Open the project in Unity.  

3. Build and run the game:  
    - Go to **File > Build Settings**, choose your target platform, and click **Build and Run**.
